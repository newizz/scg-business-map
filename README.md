# SCG Interactive Business Map

An interactive single-file HTML dashboard for visualizing SCG's official 
corporate structure (as of 1 January 2568), with deep-drill into REPCO NEX 
Industrial Solutions' full Service & Product portfolio.

## Features
- 🗺️ Hierarchical tree visualization of 8 SCG Business Units
- ⚡ One-click "Jump to REPCO" / "Jump to APM" shortcuts
- 🔍 Full-text search across all nodes
- ⭐ SET-Listed & ● Center Company badges
- ★ Visual focus highlight on Asset Performance Maximization cluster
  (DRS Platform + UHM)
- 🔗 Direct links to official REPCO NEX service pages
- 📱 Pan/Zoom/Collapse navigation
- 🌐 Single HTML file — deploys to GitHub Pages out of the box
